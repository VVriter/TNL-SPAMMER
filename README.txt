Команды:
Prefix - "."
.start - включает спаммер.
.end - выключает спаммер.
.delay {num} - ставит дилей между сообщениями в МИЛИСЕКУНДАХ, (для тупых, 1секунда = 1000 милисекунд). Команду нужно писать без "{}", пример -> .delay 60000. Рекомендованый дилей для орг ру - 60000.
